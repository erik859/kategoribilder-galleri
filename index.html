🔗 GitHub-anslutning

Ange din GitHub Personal Access Token (PAT) med repo-behörighet. Den sparas bara i din webbläsare och används för att läsa och spara galleriedata.

Skapa token på: github.com → Settings → Developer settings → Personal access tokens → Tokens (classic) → scope: repo
